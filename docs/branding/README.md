# Branding Assets

Branding assets are grouped by use. Keep new files in the narrowest folder that matches how the asset is meant to be used.

## Folders

- `palettes/` - PDF color palette references.
- `logos/team/` - team or creator logo exports.
- `logos/platforms/` - Xbox and PlayStation marks.
- `flags/` - country flag references.
- `icons/archetypes/` - source SVGs for player archetype icons.
- `icons/hockey/` - generic hockey source SVGs.
- `icons/archive/duplicates/` - exact duplicate files kept for traceability instead of deleted.
- `rink-event-map/` - rink diagram and shot, hit, goal, and penalty marker art.

## Naming

- Use lowercase kebab-case filenames.
- Drop source-site suffixes like `svgrepo-com`.
- Keep source and variant meaning in the filename when it matters, such as `hockey-box-alt.svg`.
- Do not store new loose files at the root of this folder.
